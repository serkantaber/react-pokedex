## To Install on your Computer

Clone the Project

```bash
  git clone https://github.com/serkantaber/react-pokedex
```

Go the the Project folder

```bash
  cd react-pokedex
```

Install dependencies

```bash
  npm install
```

And run the local server

```bash
  npm run start
```
